# F1 — Zvuk + Light/Dark tema

**Spec (Projektni zadatak):** zvuk kada se aktivira alarm (isključiti kad se prihvati), jačina preko slajdera, izbor zvučnog signala; dve teme: light i dark.

## Dizajn
- **Zvuk:** `AlarmSound` (static, `MediaPlayer`) — loop dok postoji `Active` alarm; `Volume` (0..1), `SoundPath` (izbor). MainWindow ima `DispatcherTimer` (1s) koji proverava `Any alarm.State == Active` → `Start()`, inače `Stop()`. Acknowledge → stanje postaje `Acknowledged` → prestaje zvuk (kako spec traži). Izbor zvuka = Windows sistemski WAV (`C:\Windows\Media\...`).
- **Tema:** `ThemeManager.Apply(dark)` swap-uje `ResourceDictionary` (Themes/Light.xaml, Dark.xaml) u `App.Resources`; implicitni stilovi za `Window`/`TextBlock` menjaju pozadinu/tekst. Default Light (u `App.OnStartup`).
- **UI:** `SettingsWindow` (dugme **Opcije**): Light/Dark radio, combo zvuka, slajder jačine, Testiraj/Stop.

- Tema menja pozadinu prozora i boju teksta; dugmad/grid ostaju default (dovoljno za light/dark efekat; može se proširiti).
